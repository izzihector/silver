Product Pricing Access Permision

Key Highlights

  • Specific Groups
  • Allow to see product price and cost
  • Overview
  • Configuration
  • Screenshots

Overview

Add groups group_product_cost & group_product_price for user

You can only see product cost or product price when you have permission

Configuration

  • Clone addon from git (extract if needed)
  • Upload addon to odoo addons folder
  • Restart odoo service
  • Go to Odoo => Apps => Update Apps List if needed
  • Search "Product Pricing Access Permision"
  • Install app
