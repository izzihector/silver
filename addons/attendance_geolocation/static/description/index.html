Attendance Geolocation

This module helps you to log the employees attendance check in / check out location information and create a direct links to Google Maps.

Features

- Makes use of the standard HTML5 geolocation API, so it works perfectly with all modern browsers, including mobile ones.
- Works better with terminals equiped with GPS (mobile phones/tablets), gets better precision.
- This module does not require GoogleMaps API KEY. Doesn't rely on Google API, so no hidden cost added. Uses standard HTML5 browser Geolocation API.
- GEOLOCATION API MAY ONLY WORKS WITH HTTPS CONNECTIONS. So your Odoo instance must be configured in https mode.



Installation

- Copy attendance_geolocation module to addons folder

- Install the module normally like other modules
